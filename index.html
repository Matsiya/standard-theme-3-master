Welcome to the Standard Theme 3 wiki! I've gone through the Standard Theme support forums and have cloned as much information as I can for this wiki. Click on the Pages tab at the top to see all of the documentation.

Note: I did not work for 8BIT or Standard Theme. I'm a user just like you that really likes Standard and want to do my part to help keep it alive. If you are interested in helping to maintain the code base and improve it, let me know.

Things I'd like to eventually do:

  • Update the core to Bootstrap 3.
  • Add support for all WordPress core post formats.
  • Change the name if this is able to be monetized through Child Themes. Core would likely remain free.
  • Replace core social media icons with larger icons.
  • Replace Standard Theme logos in Ad Widgets.
  • Add new Page Templates.
  • Add additional Widget areas.
  • Replace image file icons with Glyphicon or Font Awesome font icons.

Want to help make some of this happen?

  • This project needs someone with better PHP skills than I have.
  • This project needs someone with JavaScript skills (I have none).
  • Submit ideas for page templates or submit completed page templates you've made.
  • Submit new child themes.
  • Fork the project and submit fixes if you know of any bugs.

What I won't be doing right now.

  • Providing direct support. If you need help with something, you're on your own right now. I'll be glad to help out or do custom site design or updates if you want to hire me.
